# exchange-rate-calculator

An interactive Exchange Rate Calculator built with HTML, CSS, and JavaScript. The app allows users to convert currencies with live rates fetched from a public API (API key removed for security). This project demonstrates API integration, dynamic DOM manipulation, and responsive UI design.
